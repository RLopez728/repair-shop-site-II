# Computer Repair Shop Site

A simple landing page I built for a computer repair service concept.  
The goal was to make something clean, easy to navigate, and responsive across devices without overcomplicating the design.

## [Live Site](https://hundredplusrepairs.com)

## About
This started as a way to practice building a full landing page from scratch instead of isolated components. 
Built with a mobile-first modality, I focused on keeping the layout straightforward and making sure it looked good on both mobile and desktop. 

## What it does
- Responsive layout that adapts to different screen sizes  
- Basic navigation with interactive elements  
- Structured sections for services and information  
- Includes a WIP page for future expansion

## Built with
- HTML  
- CSS (Flexbox + media queries)  
- JavaScript

## Notes
The project is still evolving. Some sections are works in progress, and I plan to keep refining the layout and adding more features over time.

## Running it locally
Clone the repo and open `index.html` in your browser.
